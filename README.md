# from-pixels-to-code
Partial implementation of a hybrid framework for engineering drawing parsing, topology construction, and code generation.

# Paper Companion Repository

This repository is a companion package for the manuscript in this workspace. It collects a small, inspection-oriented subset of the full project so readers can review the main ideas without access to the complete engineering environment, full datasets, or full reproduction assets.

## Contents

- `docs/`: short notes and available figures for the method and system
- `rtdetr_part/`: selected RT-DETR configs, modules, demo scripts, and summarized results
- `sama_to_xml_part/`: selected SAMA-to-XML engineering code excerpts and code-generation logic
- `dataset_examples/`: a few sample images and labels plus class definitions

## Notes

- This is not the full production repository.
- Full datasets, full training assets, complete third-party runtime dependencies, and full experimental environments are intentionally omitted.
- `docs/ui_demo.png` and `sama_to_xml_part/docs/gui_screenshot.png` were not created here because no existing GUI screenshot source file was found in the workspace.

## Source Mapping

- RT-DETR source: `RTDETR-main/`
- SAMA engineering source: `SAMAToXML/`
