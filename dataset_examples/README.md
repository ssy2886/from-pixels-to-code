# Dataset Examples

This directory contains only a few sample images and labels copied from the available workspace data.

## Purpose

- Show the expected image/label format
- Show the class naming used in the PIDV5 configuration
- Avoid publishing the full dataset

## Included

- `sample_images/`: a few example images
- `sample_labels/`: a few corresponding label files
- `classes.txt`: class names extracted from the PIDV5 dataset configuration

These samples are for format inspection only and are not intended to reproduce the full training results.
